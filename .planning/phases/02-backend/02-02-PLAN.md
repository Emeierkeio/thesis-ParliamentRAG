<objective>
Rewrite the dependency injection layer from an untyped dict to typed FastAPI Depends() functions, and eliminate duplicate Neo4j connection pools in search.py and evidence.py.

Purpose: The current `get_services() -> dict` pattern has no type safety, initializes all services even when only one is needed, and cannot be overridden in tests without monkey-patching. This plan replaces it with the standard FastAPI Depends + @lru_cache pattern, making the codebase testable and type-safe. It also fixes the duplicate connection pool problem (SVC-05).

Output: Rewritten deps.py with typed functions, search.py and evidence.py using shared client, tests for DI behavior.
</objective>

Rewrite `backend/app/services/deps.py` completely. The new file must:

1. Import `lru_cache` from `functools`, `Depends` from `fastapi`
2. Import service classes: `Neo4jClient`, `RetrievalEngine`, `AuthorityScorer`, `IdeologyScorer`, `GenerationPipeline`
3. Import `get_settings` from `..config`

4. Define these functions, each decorated with `@lru_cache()`:

```python
@lru_cache()
def get_neo4j_client() -> Neo4jClient:
    settings = get_settings()
    return Neo4jClient(
        uri=settings.neo4j_uri,
        user=settings.neo4j_user,
        password=settings.neo4j_password,
    )

@lru_cache()
def get_retrieval_engine() -> RetrievalEngine:
    return RetrievalEngine(get_neo4j_client())

@lru_cache()
def get_authority_scorer() -> AuthorityScorer:
    return AuthorityScorer(get_neo4j_client())

@lru_cache()
def get_ideology_scorer() -> IdeologyScorer:
    return IdeologyScorer(get_neo4j_client())

@lru_cache()
def get_generation_pipeline() -> GenerationPipeline:
    return GenerationPipeline()
```

5. KEEP a backward-compatible `get_services()` function that calls the individual getters — this prevents breaking all routers at once. Routers will be migrated to individual Depends() in later plans. The function body becomes:
```python
def get_services() -> dict:
    """Backward-compatible wrapper. Prefer individual Depends() functions."""
    return {
        "neo4j": get_neo4j_client(),
        "retrieval": get_retrieval_engine(),
        "authority": get_authority_scorer(),
        "ideology": get_ideology_scorer(),
        "generation": get_generation_pipeline(),
    }
```
