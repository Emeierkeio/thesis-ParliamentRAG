```python
import os
import pytest
from neo4j import GraphDatabase

NEO4J_URI = os.environ.get("NEO4J_URI", "bolt://localhost:7689")
NEO4J_USER = os.environ.get("NEO4J_USER", "neo4j")
NEO4J_PASSWORD = os.environ.get("NEO4J_PASSWORD", "thesis2026")

pytestmark = pytest.mark.integration

@pytest.fixture(scope="module")
def driver():
    d = GraphDatabase.driver(NEO4J_URI, auth=(NEO4J_USER, NEO4J_PASSWORD))
    yield d
    d.close()

@pytest.fixture(scope="module")
def session(driver):
    with driver.session() as s:
        yield s
```

Test functions:

1. `test_vote_nodes_exist(session)` — `MATCH (v:Vote) RETURN count(v) AS cnt` — assert cnt > 0.

2. `test_vote_linked_to_session(session)` — `MATCH (s:Session)-[:HAS_VOTE]->(v:Vote) RETURN count(v) AS cnt` — assert cnt > 0. Also verify NO `(:Debate)-[:HAS_VOTE]->(:Vote)` edges exist.

3. `test_discusses_edges_exist(session)` — `MATCH (d:Debate)-[:DISCUSSES]->(a:ParliamentaryAct) RETURN count(*) AS cnt` — assert cnt > 0.

4. `test_speaking_role_populated(session)` — `MATCH (sp:Speech) WHERE sp.speakingRole IS NOT NULL RETURN count(sp) AS cnt` — assert cnt > 0.

5. `test_phase_type_populated(session)` — `MATCH (p:Phase) WHERE p.phaseType IS NOT NULL RETURN count(p) AS cnt` — assert cnt > 0.

6. `test_no_italian_properties(session)` — Run queries to verify dead properties are gone:
   - `MATCH (c:Chunk) WHERE c.startCharRaw IS NOT NULL RETURN count(c) AS cnt` — assert cnt == 0
   - `MATCH (c:Chunk) WHERE c.endCharRaw IS NOT NULL RETURN count(c) AS cnt` — assert cnt == 0
   - `MATCH (sp:Speech) WHERE sp.preprocessedText IS NOT NULL OR sp.preprocessed_text IS NOT NULL RETURN count(sp) AS cnt` — assert cnt == 0
   - `MATCH (s:Session) WHERE s.completeDate IS NOT NULL OR s.complete_date IS NOT NULL RETURN count(s) AS cnt` — assert cnt == 0

7. `test_no_italian_labels(session)` — `CALL db.labels()` — assert "Seduta" not in labels, "Dibattito" not in labels, "Fase" not in labels, "Intervento" not in labels, "Votazione" not in labels.

8. `test_session_has_date_property(session)` — `MATCH (s:Session) WHERE s.date IS NOT NULL RETURN count(s) AS cnt` — assert cnt > 0. Verify date is a Neo4j Date type.

9. `test_chunk_minimal_properties(session)` — `MATCH (c:Chunk) RETURN keys(c) AS k LIMIT 1` — assert keys contain only: id, text, index, embedding (embedding set later by precalculate_embeddings).
