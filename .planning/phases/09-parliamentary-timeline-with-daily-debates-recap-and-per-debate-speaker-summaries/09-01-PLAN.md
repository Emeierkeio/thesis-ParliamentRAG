Create `build/generate_summaries.py` — standalone async Python script that reads Session/Debate/Speech data from Neo4j, generates AI summaries via OpenAI gpt-4.1-mini, and writes them back as node properties.

**CLI interface:**
- `--neo4j-uri` (default `bolt://localhost:7689`)
- `--neo4j-user` (default `neo4j`)
- `--neo4j-password` (default `thesis2026`)
- `--dry-run` flag: count pending sessions/debates/speakers, estimate tokens via `len(text) / 4`, print estimated cost, exit without writing
- `--concurrency` (default `10`): max parallel OpenAI calls via `asyncio.Semaphore`

**Processing flow:**
1. Query all sessions WHERE `s.recapIt IS NULL` (resumability — skip already-done sessions)
2. For each session:
   a. Gather all debate titles for the session
   b. Generate session recap (IT + EN) via two parallel `gpt-4.1-mini` calls with `max_tokens=200`
   c. Write `Session.recapIt` and `Session.recapEn` via `SET s.recapIt = $recap_it, s.recapEn = $recap_en`
3. For each debate in each session:
   a. Gather all speech texts for the debate (ordered by phase, then speech order)
   b. If fewer than 3 speeches: skip AI summary (leave recapIt/recapEn null)
   c. Generate debate recap (IT + EN) — 3-5 sentences, two parallel calls, `max_tokens=300`
   d. Write `Debate.recapIt` and `Debate.recapEn`
4. For each (speaker, debate) pair with speeches:
   a. Combine all speech texts for that speaker in that debate
   b. If total text < 100 chars: skip (too short for meaningful summary)
   c. Generate speaker position summary (IT + EN) — 2-3 sentences
   d. Create `SpeakerDebateSummary` node via MERGE:
      ```cypher
      MERGE (sds:SpeakerDebateSummary {id: $id})
      SET sds.summaryIt = $summary_it,
          sds.summaryEn = $summary_en,
          sds.speechCount = $speech_count,
          sds.phases = $phases,
          sds.partySnapshot = $party_snapshot,
          sds.debateId = $debate_id,
          sds.speakerId = $speaker_id
      ```
      where `id = f"{debate_id}_{speaker_id}"`
   e. Link to speaker: Use `OPTIONAL MATCH (d:Deputy {id: $speaker_id}) OPTIONAL MATCH (g:GovernmentMember {id: $speaker_id}) WITH coalesce(d, g) AS sp` pattern, then `MERGE (sp)-[:HAS_DEBATE_SUMMARY]->(sds)`
   f. Link to debate: `MATCH (d:Debate {id: $debate_id}) MERGE (sds)-[:FOR_DEBATE]->(d)`

**OpenAI prompts:**
- Session recap IT: "Scrivi un riassunto di 2-3 frasi della sessione parlamentare del {date}. Argomenti trattati: {debate_titles}. Il riassunto deve essere in italiano, chiaro e conciso."
- Session recap EN: "Write a 2-3 sentence summary of the parliamentary session of {date}. Topics discussed: {debate_titles}. The summary must be in English, clear and concise."
- Debate recap IT: "Scrivi un riassunto di 3-5 frasi del seguente dibattito parlamentare: '{title}'. Testi degli interventi: {speech_excerpts}. Il riassunto deve essere in italiano e coprire i punti principali."
- Debate recap EN: Same but in English.
- Speaker summary IT: "Scrivi un riassunto di 2-3 frasi della posizione di {speaker_name} ({party}) nel dibattito '{debate_title}'. Testi: {speech_texts}."
- Speaker summary EN: Same but in English.

For debate/speaker prompts, truncate combined speech text to 4000 chars to stay within context limits.

**Progress and logging:**
- Use `tqdm` progress bar for sessions
- Log: `logger.info(f"Session {date}: {n_debates} debates, {n_speakers} speaker summaries")`
- Log total at end: sessions processed, debates processed, speaker summaries created, total API calls, elapsed time

**Rate limiting:**
- Use `asyncio.Semaphore(concurrency)` to cap parallel OpenAI calls
- Use `openai.AsyncOpenAI` with `max_retries=3` (built-in retry handles 429/5xx)

**Dry run mode:**
- Count sessions without recapIt, debates without recapIt, (speaker, debate) pairs
- Estimate tokens: `sum(len(text) / 4 for text in all_texts)`
- Print: "Pending: {n} sessions, {n} debates, {n} speaker summaries"
- Print: "Estimated tokens: {n}, Estimated cost: ${n:.2f} (at $0.40/1M input + $1.60/1M output)"
- Exit without writing

```makefile
# ============================================================================
#  AI Summary Generation

.PHONY: generate-summaries db-full

generate-summaries: db-install ## Generate AI summaries for timeline (resumable)
	@printf "$(BOLD)$(CYAN)Generating AI summaries for timeline...$(RESET)\n"
	@docker compose up -d neo4j
	@printf "$(CYAN)Waiting for Neo4j bolt port (7689)...$(RESET)\n"
	@for i in $$(seq 1 30); do \
		$(PYTHON) -c "from neo4j import GraphDatabase; d=GraphDatabase.driver('$(NEO4J_LOCAL)',auth=('$(NEO4J_USER)','$(NEO4J_PASS)')); s=d.session(); s.run('RETURN 1').single(); s.close(); d.close()" 2>/dev/null && break; \
		printf "."; \
		sleep 3; \
	done
	@printf "\n$(GREEN)Neo4j ready$(RESET)\n"
	@$(PYTHON) $(BUILD_DIR)/generate_summaries.py \
		--neo4j-uri $(NEO4J_LOCAL) \
		--neo4j-user $(NEO4J_USER) \
		--neo4j-password $(NEO4J_PASS) \
		$(if $(DRY_RUN),--dry-run,)
	@printf "\n$(BOLD)$(GREEN)Summary generation complete!$(RESET)\n"

db-full: db-all generate-summaries ## Full DB build + AI summaries (one-shot)
```
