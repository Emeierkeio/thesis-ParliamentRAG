<!-- SPARQL endpoint vocabulary (verified live during research) -->

Individual vote record (ocd:voto):
- URI pattern: http://dati.camera.it/ocd/voto.rdf/vd19_{session}_{vote}_{personId}
- ocd:rif_deputato -> deputato.rdf/d{personId}_19
- ocd:rif_votazione -> votazione.rdf/vs19_{session}_{vote}
- ocd:type -> "Favorevole" | "Contrario" | "Astenuto" | "Non ha votato" | "In missione"

Committee officer role (ocd:ufficioParlamentare):
- ocd:carica -> "PRESIDENTE" | "VICEPRESIDENTE" | "SEGRETARIO" | ...
- ocd:rif_organo -> organo.rdf/o19_{id} (rdfs:label = committee name)
- ocd:rif_deputato -> deputato.rdf/d{personId}_19
- ocd:startDate -> "YYYYMMDD"
- ocd:endDate -> "YYYYMMDD" (may be absent)

Deputy ID mapping:
- SPARQL: deputato.rdf/d{personId}_19 -> extract numeric {personId}
- Neo4j: Deputy.id = "http://dati.camera.it/ocd/persona.rdf/p{personId}"

Vote matching:
- SPARQL votazione URI: .../vs19_{session}_{vote}
- Neo4j: Session.number = {session}, Vote.number = {vote}
- Cypher: MATCH (s:Session {number: $session_num})-[:HAS_VOTE]->(v:Vote {number: $vote_num})

From build/db_builder.py:
```python
class DatabaseBuilder:
    def _batch_write(self, neo_session, fn, items: list) -> None:
        """Split items into batches and call fn via execute_write for each."""
        batch_size = self._config.batch_size
        for i in range(0, len(items), batch_size):
            neo_session.execute_write(fn, items[i:i + batch_size])
```

    Create `build/sparql_ingester.py`:

    1. Module-level constants:
       - `SPARQL_ENDPOINT = "https://dati.camera.it/sparql"`
       - `OUTCOME_MAP` dict mapping Italian vote types to English (favor, against, abstain, absent, on_mission)
       - `BATCH_SIZE = 500` for Neo4j writes
       - `SPARQL_PAGE_SIZE = 1000` for HTTP pagination
       - `SPARQL_TIMEOUT = 30` seconds

    2. URI parsing helpers (pure functions, easily testable):
       - `sparql_dep_uri_to_neo4j_id(uri: str) -> Optional[str]`: Extract numeric ID from `d{id}_19` pattern, return `persona.rdf/p{id}` full URI. Return None if pattern doesn't match.
       - `parse_votazione_uri(uri: str) -> tuple[Optional[int], Optional[int]]`: Extract (session_number, vote_number) from `vs19_{session}_{vote}` pattern.

    3. SPARQL HTTP helper:
       - `_sparql_get(query: str, timeout: int = SPARQL_TIMEOUT) -> list[dict]`: URL-encode query, GET to endpoint with `format=application/sparql-results+json`. Wrap in try/except for timeout and HTTP errors. Return empty list on failure. Use `urllib.request.urlopen`.

    4. Class `SparqlIngester`:
       - `__init__(self, driver)`: Store neo4j driver reference.
       - `ingest_votes(self, limit_deputies: int = 0) -> dict`: Main vote ingestion.
         a. Query Neo4j for all Deputy nodes: `MATCH (d:Deputy) RETURN d.id AS id`
         b. For each deputy, extract numeric person_id from Deputy.id (`persona.rdf/p{id}`)
         c. Build SPARQL deputato URI: `deputato.rdf/d{person_id}_19`
         d. Page through `get_deputy_votes(dep_uri, offset)` with LIMIT 1000
         e. For each vote binding: parse votazione URI to get (session_num, vote_num), map outcome, build batch dict
         f. Batch write to Neo4j with Cypher:
            ```
            UNWIND $batch AS row
            MATCH (d:Deputy {id: row.deputyId})
            MATCH (s:Session {number: row.sessionNumber})-[:HAS_VOTE]->(v:Vote {number: row.voteNumber})
            MERGE (iv:IndividualVote {id: row.id})
            SET iv.outcome = row.outcome
            MERGE (d)-[:VOTED]->(iv)
            MERGE (iv)-[:ON_VOTE]->(v)
            ```
         g. IndividualVote.id format: `iv_{person_id}_{session}_{vote}`
         h. Log progress: deputy count, votes written, votes skipped (no matching Vote node)
         i. If `limit_deputies > 0`, only process that many deputies (for testing)
         j. Return stats dict: `{"deputies_processed": N, "votes_written": N, "votes_skipped": N}`

       - `ingest_committee_roles(self) -> dict`: Committee officer role enrichment.
         a. Query Neo4j for all Deputies (same as above)
         b. For each deputy, fetch committee officer roles via SPARQL query
         c. For each role binding: extract committeeName from organoLabel, role from carica, dates from startDate/endDate
         d. Write to Neo4j with Cypher:
            ```
            UNWIND $batch AS row
            MATCH (d:Deputy {id: row.deputyId})-[r:MEMBER_OF_COMMITTEE]->(c:Committee)
            WHERE toLower(c.name) CONTAINS toLower(row.committeeName)
            SET r.officerRole = row.role,
                r.officerRoleStart = date(row.startDate),
                r.officerRoleEnd = CASE WHEN row.endDate IS NOT NULL THEN date(row.endDate) ELSE NULL END
            ```
         e. Return stats dict

       - `_get_deputy_votes_page(self, dep_sparql_uri: str, offset: int = 0) -> list[dict]`: Build and execute SPARQL query for one page of votes.
       - `_get_committee_roles(self, dep_sparql_uri: str) -> list[dict]`: Build and execute SPARQL query for committee roles.

       if __name__ == "__main__":
